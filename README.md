# Autolandia

Приложение для автоcервиса. Есть программа для менеджеров и сайт для клиентов.

## Что тут есть

- **Сайт для клиентов** (Vue.js) - клиенты смотрят машины и оформляют заказы
- **Программа для менеджеров** (WPF) - менеджеры и бухгалтеры работают с заказами
- **Сервер** (ASP.NET Core API) - общий для сайта и программы
- **База данных** (MySQL) - где все хранится

## Что нужно установить сначала

1. **.NET 8.0** - отсюда: https://dotnet.microsoft.com/download/dotnet/8.0
2. **Node.js** - отсюда: https://nodejs.org/ (берите версию 16 или новее)
3. **MySQL** - отсюда: https://dev.mysql.com/downloads/mysql/
4. **Git** - если еще нет

## Как все запустить

### 1. Скачиваем проект

```bash
git clone [ваша ссылка на репозиторий]
cd autolandia
```
2. Настраиваем базу данных
Запустите MySQL и создайте базу:

```sql
CREATE DATABASE autolandia;
Если есть файл с данными (например, backups/database.sql), загрузите его:
```

```bash
mysql -u root -p autolandia < backups/database.sql
```
3. Генерируем модели для базы данных
Зайдите в папку с API и выполните команду:

```bash
Scaffold-DbContext "Server=127.0.0.1;User=root;Password=ваш_пароль;Database=autolandia" Pomelo.EntityFrameworkCore.MySql -OutputDir "Models" -Force
Замените root и ваш_пароль на свои
```

4. Запускаем сервер (API)
```bash
cd API
dotnet run
Сервер запустится на https://localhost:7000
```

5. Запускаем сайт для клиентов
Откройте новый терминал:

```bash
cd vue-project
npm install
npm run dev
Сайт откроется на http://localhost:3000
```

6. Запускаем программу для менеджеров
Откройте файл WPF-Client/Autolandia.sln в Visual Studio и нажмите F5

Что где будет работать
Сайт для клиентов → ваша ссылка

Программа для менеджеров → запускается на компьютере

Сервер API → ваша ссылка

База данных → ваша ссылка

Если что-то не работает
База данных не подключается
Проверьте, что MySQL запущен

Проверьте логин и пароль

Убедитесь, что база autolandia создана

Сайт не запускается
Попробуйте удалить папку node_modules и выполнить npm install снова

Проверьте, что Node.js установлен

Сервер не работает
Убедитесь, что .NET 8.0 установлен

Проверьте строку подключения к базе в appsettings.json

Папки в проекте

autolandia/

├── backups/            # Резервные копии базы
└── docs/               # Документы
